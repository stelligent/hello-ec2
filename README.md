# Fork and clone this repo
```
git clone git@github.com:stelligent/hello-ec2.git
```

# Install mu
```
curl -s https://getmu.io/install.sh | sh
```

## on AWS Cloud9
```
sudo sh
curl -s https://getmu.io/install.sh | sh
exit
```

# Create the pipeline
```
mu pipeline up -t [GitHub Personal Access Token]
```
